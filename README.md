# flat-me

> The easiest way to find your next apartment

### Setup

```bash
git clone https://github.com/zsevic/flat-me
cd flat-me
cp .env.sample .env.local # change values after copying
npm i
npm run dev
```

### Build

```bash
npm run build
npm start
```

### Linting

```bash
npm run lint
npm run lint:fix
```

### Testing

```bash
npm test
```

### Technologies used

- React, Next.js
